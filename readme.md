# Tumblr Themes by sanographix

## View All Themes

## Overview

### GitHub 版（ここに置いてあるやつ）

- たまにデザインを修正・変更します
- 各テーマのhtmlファイルをコピーしてお使いください

### テーマガーデン版

- Tumblrから直接インストールできるようにしたものです
- ここからインストールできます
    - <https://www.tumblr.com/themes/by/sanographix-theme>
- テーマガーデン版は、Tumblrのテーマガイドラインに適合させるため、GitHub版より仕様を削っています。
